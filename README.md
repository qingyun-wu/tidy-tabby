# Tidy Tabby 🐱

**Tidy Tabby keeps your tabs tidy.**

Tidy Tabby is a Chrome extension that turns your new tab page into a personal dashboard — open tabs, browsing history, bookmarks, and an AI assistant, all in one place.

---

## Features

### Open Tabs
- Tabs grouped by domain with color-coded staleness indicators (active / cooling / stale)
- Chrome Tab Groups view — switch between Domains and Groups with one click
- Search bar with `/` keyboard shortcut to filter tabs instantly
- Pinned tabs excluded from the dashboard (they stay where they are)
- Duplicate tab detection with one-click cleanup
- Close individual tabs or entire groups with swoosh + confetti
- Save tabs for later to a checklist sidebar
- Native favicons from Chrome, no external API calls

### Recent Activity
- Browsing history with Today / This week / This month range selector
- **By site** view — top domains ranked by visits with activity bars
- **By session** view — timeline grouped into browsing sessions
- "Away from browser" gaps shown between sessions (green indicators)
- Per-day summary: browsing time vs. away time at a glance
- Idle state disclaimer when session times may be inaccurate
- AI-powered history summary with one click

### Bookmarks
- Quick Access bar — top 10 items from your bookmarks bar, folders expand inline
- Organized by folder with search
- Duplicate bookmarks banner — one-click cleanup, keeps one copy of each
- Old bookmarks banner — archive bookmarks over a year old into year folders (Archive/2020, Archive/2021, etc.)
- Move bookmarks between folders with auto-suggested folder names
- AI-powered organization suggestions via Claude

### Tabby Chat
- AI assistant aware of your open tabs, browsing history, and current page
- Privacy mode integration — ask AI directly from the lock screen
- Quick suggestion prompts in the new tab page
- Conversation history maintained per session

### Side Panel
- Lightweight tab list + Tabby Chat in the browser sidebar
- Click the toolbar icon to open
- Auto-summarizes the current page when you switch tabs
- Clickable follow-up suggestions generated by AI
- Reads actual page content for accurate summaries
- Search and close tabs without leaving the current page

### Privacy Mode
- One-click full-screen cover (toggle with Esc or lock icon)
- Clock, date, and customizable text
- AI assistant built into the privacy screen
- All dashboard content completely hidden

### Quality of Life
- Dark mode — follows system preference automatically
- Auto-refresh — dashboard updates live when tabs change
- Tabbed layout — Open Tabs, Recent Activity, Bookmarks, Tabby Chat all accessible without scrolling
- 100% local storage — no accounts, no server, no tracking
- AI features are optional, powered by your own Anthropic API key

## Setup

**1. Clone the repo**

```bash
git clone https://github.com/qingyun-wu/tidy-tabby.git
cd tidy-tabby
```

**2. Load the Chrome extension**

1. Open Chrome → `chrome://extensions`
2. Enable **Developer mode** (top-right)
3. Click **Load unpacked** → select the `extension/` folder

**3. Open a new tab** — you'll see Tidy Tabby.

---

## AI Features

Tabby Chat and AI-powered features require an [Anthropic API key](https://console.anthropic.com/). Enter it in the Tabby Chat tab — stored locally, never sent anywhere except Anthropic's API.

## Tech stack

| What | How |
|------|-----|
| Extension | Chrome Manifest V3 |
| Storage | chrome.storage.local |
| AI | Anthropic Claude API (optional) |
| Side Panel | Chrome Side Panel API |
| Sound | Web Audio API (synthesized) |
| Animations | CSS transitions + JS confetti |
